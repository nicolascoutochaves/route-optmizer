Como usar o Otimizador de Roteiros

O programa lê seus arquivos KML, geocodifica os endereços de cada parada e calcula a rota mais curta partindo e retornando sempre à base fixa. Este guia resume cada etapa — sem enrolação — para consulta enquanto você trabalha.

Referência rápida

01

Importar roteiros

  • Arraste um ou mais arquivos .kml exportados do Google Maps até a área de upload.
  • Vários arquivos podem ser carregados de uma vez.
  • Os dados ficam salvos automaticamente no navegador após o carregamento.
02

Vincular um arquivo JSON

  • 🔗 Importar json liga o programa a um arquivo no seu computador.
  • 🔄 Recarregar busca as alterações mais recentes desse arquivo.
  • 🔌 Desvincular apenas desliga a ligação — o arquivo não é apagado.
03

Selecionar um roteiro

  • Clique em um cartão na tela inicial para abrir a lista de pontos.
  • Clique em qualquer ponto da lista para corrigir endereço ou coordenada manualmente.
  • Arraste as linhas da lista para reordenar as paradas na mão.
04

Editar pontos em detalhe

  • O botão ✏️ Editar abre um formulário por ponto: nome, endereço, latitude e longitude.
  • ➕ Adicionar ponto cria uma parada nova com coordenadas digitadas na mão.
  • 💾 Salvar alterações grava no roteiro e, se houver JSON vinculado, no próprio arquivo.
05

Gerar link × Otimizar

  • 🔗 Gerar Link do Maps monta o trajeto na ordem atual da lista.
  • ⚡ Geocodificar + Otimizar além de geocodificar, reordena tudo pela rota mais curta.
  • Use "Gerar Link" quando a ordem das paradas já importa para você.
06

Roteiro personalizado

  • O cartão ✏️ Personalizar monta uma rota escolhendo pontos de vários roteiros já carregados.
  • Marque os pontos desejados e organize a ordem antes de gerar o link.
  • Dá para salvar esse roteiro personalizado para reaproveitar depois.
07

Compartilhar a rota

  • 🗺️ Google Maps abre o trajeto calculado direto no navegador.
  • 📋 Copiar link ou o QR Code exibido facilitam enviar para o celular.
  • 🔄 Gerar link na tela de resultado atualiza o link após reordenar pontos.
08

Exportar e fazer backup

  • ⬇️ Baixar KML salva a rota otimizada como um novo arquivo KML.
  • 💾 Exportar JSON baixa uma cópia de segurança de todos os roteiros carregados.
  • Guarde essa cópia de vez em quando — ela substitui o cache do navegador se algo for perdido.

Problemas comuns

Ponto marcado ✕ Corrigir
O endereço não foi encontrado. Clique no ponto e digite um endereço mais específico (rua, número e bairro).
"Permissão negada"
Conceda o acesso quando o navegador pedir, ao clicar em 🔄 Recarregar no arquivo vinculado.
Nada aparece após carregar o KML
Confirme que o arquivo tem pastas com pontos dentro — é a estrutura padrão de uma exportação do Google Maps.
Coordenada errada mesmo após corrigir
Use o botão ✏️ Editar e digite a latitude/longitude corretas na mão, em vez de depender da geocodificação automática.
Roteiro sumiu depois de limpar o navegador
Restaure a partir de um backup com 📥 Importar roteiros.json, ou vincule sempre um arquivo JSON externo.
